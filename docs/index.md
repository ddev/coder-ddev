# DDEV Coder Workspaces

Cloud-hosted development environments for Drupal core contribution. Full DDEV stack — Drupal core clone, running site, drush — ready in **~30 seconds**. No local setup required.

[![Open in Coder](https://coder.ddev.com/open-in-coder.svg)](https://coder.ddev.com/templates/coder/drupal-core/workspace?mode=manual)

---

## How it works

1. Click the button above and sign in with GitHub
2. Click **Create Workspace** — setup runs automatically
3. Click **DDEV Web** to open your Drupal site, or **VS Code** to start editing

Admin credentials: `admin` / `admin` &nbsp;·&nbsp; One-time login: `ddev drush uli`

Drupal core lives in `~/drupal-core/repos/drupal/` — a clean git clone, ready for patches.
